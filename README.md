# 🚗 내 차 vs 네 차! 1년 유지비 배틀 시뮬레이터

안녕하세요. IT 기술과 일상의 흥미로운 주제를 탐구하는 유튜브 채널 **'잡학다식 개발자'**의 오픈소스 저장소입니다. 
본 프로젝트는 차량의 연비, 유종(고급유/일반유/경유), 자동차세, 소모품 비용을 종합하여 **'실제 1년 유지비'**를 직관적으로 비교해 보는 파이썬 기반 웹 시뮬레이터입니다.

본 프로젝트는 더 나은 기능 업데이트 방향을 설정하기 위해, 실행 시 익명화된 최소한의 사용 통계(실행 횟수, 국가/도시 등)를 수집합니다.

## 💡 주요 기능 (Features)
* **실시간 단가 반영:** 현재 주유소의 일반유, 고급유, 경유 가격을 직접 입력하여 계산에 반영할 수 있습니다.
* **차량 제원 DB 제공:** 2020 르노 클리오, 2019 BMW M2 컴페티션 등 실제 차량들의 기본 제원이 내장되어 있어 빠른 비교가 가능합니다.
* **커스텀 차량 입력:** 본인의 차량 이름과 공인 연비, 세금 등을 직접 입력하여 1:1 맞춤형 배틀을 진행할 수 있습니다.
* **직관적인 시각화:** 계산된 총 유지비와 순수 기름값의 차이를 바 차트(Bar Chart)로 한눈에 비교합니다.
